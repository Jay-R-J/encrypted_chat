# 安全加密对话工具

一个基于浏览器的端到端加密消息工具，保护您的隐私安全。

## 🌟 功能特性

### 🛡️ 核心安全特性
- **端到端加密**：使用 RSA-OAEP + AES-GCM 算法组合，确保消息安全
- **密钥本地存储**：所有密钥存储在您的设备上，不会发送到服务器
- **浏览器内加密**：所有加密解密操作在您的浏览器中完成
- **前向保密**：每次加密使用不同的会话密钥

### 📱 用户体验优化
- **直观的界面设计**：简洁明了的操作流程，易于使用
- **响应式设计**：完美适配桌面和移动设备
- **实时加载状态**：操作过程中显示优雅的加载动画
- **Toast通知**：美观的浮动通知，提供及时反馈
- **拖拽上传**：支持将文本文件拖拽到输入框
- **自动保存草稿**：防止意外刷新丢失内容

### 🗝️ 密钥管理
- **密钥对生成**：一键生成高强度RSA-4096密钥对
- **密钥导出导入**：支持备份和恢复密钥
- **密钥强度评估**：实时显示密钥安全等级
- **密钥过期提醒**：30天有效期，到期前自动提醒更换
- **公钥分享**：方便地复制和分享您的公钥

### 📋 消息处理
- **加密消息**：使用接收方公钥加密消息
- **解密消息**：使用私钥解密收到的消息
- **分享链接**：生成包含加密消息的分享链接
- **消息历史记录**：保存加密消息历史，方便查看

### 🌓 个性化设置
- **主题切换**：支持浅色/深色主题
- **多语言支持**：提供中文界面
- **可访问性优化**：支持键盘导航和屏幕阅读器

## 🚀 快速开始

### 在线使用
访问 [http://localhost:8000](http://localhost:8000) 开始使用。

### 本地部署
1. 克隆或下载项目文件
2. 在项目目录中启动HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 📖 使用指南

### 1. 密钥交换流程
1. **生成密钥**：点击「生成新密钥对」，获得您的公钥和私钥
2. **分享公钥**：复制页面上显示的「您的公钥」，通过安全渠道发送给聊天对象
3. **接收公钥**：将对方分享的公钥粘贴到「接收方公钥」文本框

### 2. 发送加密消息
1. **输入消息**：在「输入明文消息」框中输入您要发送的内容
2. **选择公钥**：确保「接收方公钥」框中已填入对方的公钥
3. **加密消息**：点击「加密消息」按钮，生成密文
4. **分享密文**：
   - 方式1：复制「加密结果」文本框中的内容，发送给对方
   - 方式2：点击「生成分享链接」，复制链接发送给对方（推荐）

### 3. 接收解密消息
1. **获取密文**：
   - 方式1：直接从对方获取密文，粘贴到「输入加密消息」框
   - 方式2：打开对方分享的链接（密文会自动填充）
2. **解密消息**：点击「解密消息」按钮
3. **查看结果**：在「解密结果」框中查看原始消息

## 🔒 安全特性

- **零知识架构**：服务器无法访问您的消息内容
- **防钓鱼警告**：检测域名安全性，防止钓鱼攻击
- **安全审计日志**：记录所有密钥操作和消息处理
- **密钥强度评估**：确保使用高强度密钥
- **自动清除机制**：支持一键清除所有密钥
